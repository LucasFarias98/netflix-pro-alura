Netflix Pro - Imersão Alura.

Uma aplicação web inspirada na experiência da Netflix, com intro cinematográfica em Vimeo, seleção de perfis, busca, favoritos, tema claro/escuro, cards de conteúdo e página de detalhes. O projeto foi desenvolvido para oferecer um visual premium, navegação fluida e funcionamento estático no GitHub Pages. Minha trajetória de aprendizado foi construída ao longo de 6 anos na Alura, onde aprendi muito com professores capacitados, uma plataforma completa e uma comunidade extremamente acolhedora e prestativa.

## Demonstração
Assim que publicado, o projeto pode ser acessado via GitHub Pages.

 Funcionalidades
- Intro com vídeo do Vimeo.
- Hover para ativar o som da intro.
- Seleção de perfis.
- Tema claro e escuro.
- Busca por perfis e conteúdos.
- Seções de conteúdos em carrossel.
- Favoritos e “Minha lista”.
- Tela de detalhes do conteúdo.
- Layout responsivo para desktop e mobile.

Tecnologias
- HTML5
- CSS3
- JavaScript
- Vimeo Embed
- GitHub Pages
- Copilot IA
